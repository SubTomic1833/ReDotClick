#  ReDotClick

A retro style game where you are given 3 lives, and one goal : click the red dots in order to survive!

## TRY IT FOR YOURSELF : https://subtomic1833.github.io/ReDotClick/

---

##  Gameplay

- A red (or sometimes blue) circle appears on the screen
- Click it before the timer hits zero
- Missing a target or clicking the wrong one costs a life
- Lose all lives = Game Over
- Reach the required score to complete each level

---

##  Levels

###  Level 1
- Standard reaction speed (1.5 second intervals)
- Reach **25 points** to win

###  Level 2
- Faster timer (1 second intervals)
- Reach **40 points** to win

###  Level 3
- Even faster reaction time (0.75 second intervals)
- Blue circles appear (DO NOT CLICK THEM, YOU WILL LOSE A LIFE)
- Reach **50 points** to win
